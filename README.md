# MyARCore: My ARCore practice demo
个人ARCore练习demo。参考资料见[官方教程](https://developers.google.com/ar/develop/java/)，及[官方文档](https://developers.google.com/ar/reference/)。

练习内容：
- 添加Android控件
- 添加3D模型
- 添加简单几何体
- 运行时动态添加模型
- 屏幕录制
- 在空中添加物体
